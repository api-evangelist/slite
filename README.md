# Slite

Slite is a collaborative documentation and AI-powered knowledge base platform that enables teams to create, organize, and maintain shared notes, documents, and team knowledge bases for asynchronous work. The platform provides a Public REST API following the OpenAPI v3.0 standard for programmatic access to workspaces, documents, users, groups, and AI search capabilities.

## API

- **Developer Portal:** https://developers.slite.com/
- **OpenAPI Specification:** https://api.slite.com/openapi.json
- **Base URL:** https://api.slite.com/v1
- **Authentication:** Personal API key via `x-slite-api-key` request header, generated under Settings > API

## Key Capabilities

- Create, update, and delete notes and documents
- Full-text and AI-powered natural language search (Ask)
- Manage channels, hierarchies, and knowledge structure
- User and group management
- Knowledge management workflows (verify, archive, flag outdated)
- MCP Server integration for AI assistant access
